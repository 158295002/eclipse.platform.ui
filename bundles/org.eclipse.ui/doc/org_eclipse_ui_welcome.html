Welcome

Identifier: org.eclipse.ui.welcome

Description: This extension point is used to add wizards to the Welcome dialog box.  Each wizard appears as a choice within the dialog box and, if selected, is created and run.

Configuration Markup:

   <!ELEMENT wizard EMPTY>
   <!ATTLIST wizard
      id     CDATA #REQUIRED
      name   CDATA #REQUIRED
      class  CDATA #REQUIRED
   >

  • id - a unique name that will be used to identify this Welcome wizard entry
  • name - a translatable name that will be used in the dialog box to represent this entry
  • class  - a fully qualified name of a class that implements org.eclipse.ui.INewWizard interface
Examples:

The following is an example of a Welcome extension definition:

   <extension point="org.eclipse.ui.welcome">
      <wizard
         id="com.xyz.quickWizard1"
         name="XYZ Wizard"
         class="com.xyz.wizards.quickWizard1">
      </wizard>
   </extension>

API Information: The class attribute should represent a fully qualified name of a class that implements org.eclipse.ui.INewWizard interface.

Supplied Implementation: The workbench provides two choices in the Welcome dialog box:

  • Go to the workbench
  • Join a team
This list is subject to change depending on such things as installation options and product version. The end user product will typically include more choices contributed by plug-ins.
